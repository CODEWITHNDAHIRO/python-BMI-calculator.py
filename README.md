**PYTHON BMI CALCULATOR:**
This is lightweight python script that calculates Body Mass Index (BMI) and provides a health category classification based on user input.
To implement the BMI formula, the script uses the standard formula BMI = weight*703/(height^2)
it features logic to categorize results into several health tiers, from "underweight" to "Morbidly obese"
Features:
1. USER PERSONALIZATION: greets the user by name in the final assessment
2. AUTOMATED CALCULATION: handles the math for BMI based on height and weight
3. HEALTH CLASSIFICATION: uses conditional logic to provide a specific weight category.
4. CLEAN CODE: simple, readable Python script ideal for beginners.

**how to use**
1. git clone https://github.com/CODEWITHNDAHIRO/python-BMI-calculator.py.git
2. cd python-BMI-calculator.py
3. run the script: python "BMI CALCULATOR.py"
